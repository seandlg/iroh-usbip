# iroh-usbip

Securely share and access USB devices over the internet with zero configuration using the power of peer-to-peer networking. By tunneling USB-over-IP traffic through Iroh's encrypted QUIC streams, `iroh-usbip` eliminates the need for complex port-forwarding, VPNs, or firewalls. It provides a lightweight, cross-platform solution for seamlessly bridging physical hardware between remote machines.
